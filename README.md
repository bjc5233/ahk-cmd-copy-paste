# ahk-console-copy-paste


## 说明
> 终端窗口原始并没有复制粘贴相关快捷键操作, 此脚本增加该特性


### cmd console
* Ctrl + A ==> 全选复制
* Ctrl + C ==> 复制最后一行命令内容
* Ctrl + V ==> 粘贴
* Ctrl + Z ==> 删除一个字符
* Ctrl + L ==> 清屏


### git bash
* Ctrl + V ==> 粘贴
* Ctrl + Z ==> 删除一个字符
* Ctrl + L ==> 清屏


## 注意
> 由于Ctrl + C在console中一般用于强制结束命令，因此转为Ctrl + K
